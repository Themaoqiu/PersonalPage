  summary={"论文提出 InterleaveThinker，这是一个多 agent 流水线，目标是让现有图像生成器具备文本-图像交错生成能力。方法上，它先用 planner agent 组织图文输入序列，再用 critic agent 检查生成结果、定位偏离计划的样本，并修正指令后重新生成。作者还构建了对应的监督数据集，并用 GRPO 强化学习提升逐步纠错能力，同时用 accuracy reward 和 step-wise reward 近似优化长轨迹。实验表明，该方法能显著提升多种图像生成器在 interleaved generation 基准上的表现，效果接近 Nano Banana 和 GPT-5。它还在 4-step FLUX.2-klein 上对 WISE 和 RISE 等推理型基准带来明显增益。"}

## 3D/Space Reasoning

<div class="space-y-4">
<PaperCard
  title={"SpatialClaw: Rethinking Action Interface for Agentic Spatial Reasoning"}
  arxivUrl={"https://arxiv.org/abs/2606.13673"}
  hjfyUrl={"https://hjfy.top/arxiv/2606.13673"}
  summary={"论文提出 SpatialClaw，用代码作为空间推理 agent 的动作接口，重新设计 VLM 调用感知与几何工具的方式。系统在状态化 Python kernel 中预加载输入帧和感知、几何原语，让 agent 能够每一步基于前一步输出写可执行代码，从而更灵活地组合操作并适应 3D/4D 空间问题。作者在 20 个覆盖静态与动态 3D/4D 空间推理的基准上进行评测，SpatialClaw 平均准确率达到 59.9%。它比近期空间 agent 高出 11.2 个百分点，并且在来自两个模型家族的 6 个 VLM backbone 上都带来稳定收益，无需针对具体基准或模型做额外适配。"}
/>
</div>

## Agent Training and Evaluation

<div class="space-y-4">
<PaperCard
  title={"EvoArena: Tracking Memory Evolution for Robust LLM Agents in Dynamic Environments"}
  arxivUrl={"https://arxiv.org/abs/2606.13681"}
  hjfyUrl={"https://hjfy.top/arxiv/2606.13681"}
  summary={"论文提出 EvoArena，用于评测 LLM agent 在动态环境中的记忆与行为适应能力。它将环境变化建模为 terminal、software 和 social 三个领域中的连续更新序列，并提出 EvoMem 这种基于补丁的记忆范式，把记忆演化过程结构化记录下来，帮助 agent 根据历史变化推理当前状态。实验显示，现有 agent 在 EvoArena 上平均准确率只有 39.6%，说明面对演化任务时表现明显不足。EvoMem 在 EvoArena 上平均提升 1.5%，并在 GAIA 和 LoCoMo 上分别提升 6.1% 和 4.8%。在需要连续完成相关子任务的 chain-level 评测中，它还带来 3.7% 的提升，机制分析表明其主要改善了证据捕获和完整状态保留。"}
/>

  arxivUrl={"https://arxiv.org/abs/2606.12087"}
