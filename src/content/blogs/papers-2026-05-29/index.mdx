## Multimodal World Model

<div class="space-y-4">
<PaperCard
  title={"Gamma-World: Generative Multi-Agent World Modeling Beyond Two Players"}
  arxivUrl={"https://arxiv.org/abs/2605.28816"}
  hjfyUrl={"https://hjfy.top/arxiv/2605.28816"}
  summary={"这篇工作提出了面向多智能体交互环境的生成式世界模型 Gamma-World，目标是在共享空间中同时建模多个可独立控制的智能体。方法上，它用 Simplex Rotary Agent Encoding 让不同智能体在旋转位置编码空间中保持可区分但又排列等价，并用 Sparse Hub Attention 通过可学习的 hub token 将跨智能体交互从二次复杂度降到线性。为了支持实时 rollout，作者还把全上下文扩散教师蒸馏成因果学生模型，按时间块顺序生成并结合 KV cache。实验证明，该模型在多玩家虚拟环境中比基于 slot 和 dense attention 的基线具有更好的视频质量、动作可控性和智能体一致性，并且无需额外训练即可从两人泛化到四人。"}
/>
</div>
