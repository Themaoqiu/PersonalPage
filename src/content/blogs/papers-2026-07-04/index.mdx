<PaperCard
  title={"AgenticSTS: A Bounded-Memory Testbed for Long-Horizon LLM Agents"}
  arxivUrl={"https://arxiv.org/abs/2607.02255"}
  hjfyUrl={"https://hjfy.top/arxiv/2607.02255"}
  summary={"这篇工作提出 AgenticSTS，一个面向长程 LLM Agent 的有界记忆评测基准，用来研究“记忆层如何影响后续决策”。作者把每一步决策都构造成新的用户消息，并通过类型化检索组装上下文，从而避免把完整历史对话直接拼进提示词。基准落在 Slay the Spire 2 这类高难度、随机且需要大量连续决策的环境中，并发布了 298 条完整轨迹、冻结的记忆/技能快照和分析脚本，便于复现。实验显示，在该测试框架下，启用策略技能层后胜率从 3/10 提升到 6/10，但样本规模下差异仍偏方向性，未达到统计显著。"}

<PaperCard
  title={"EvoPolicyGym: Evaluating Autonomous Policy Evolution in Interactive Environments"}
  arxivUrl={"https://arxiv.org/abs/2607.02440"}
  hjfyUrl={"https://hjfy.top/arxiv/2607.02440"}
  summary={"这篇工作提出 EvoPolicyGym，用来评测自主 Agent 在交互式环境中如何在固定预算内反复修改可执行策略并逐步变好。作者把“自主策略进化”定义成一个受控评测设置，避免把它和泛化的软件工程进展混在一起。基准由多个紧凑的交互式强化学习环境组成，并提供轨迹级诊断来分析 Agent 如何分配预算、如何把反馈转化为参数调整。实验表明，GPT-5.5 在综合排名上表现最好，并在 16 个环境里都进入前二，说明该任务不仅看单次解题能力，也考验在有限反馈下持续改进策略的能力。"}
/>

<PaperCard
  title={"AgenticDataBench: A Comprehensive Benchmark for Data Agents"}
  arxivUrl={"https://arxiv.org/abs/2607.01647"}
  hjfyUrl={"https://hjfy.top/arxiv/2607.01647"}
  summary={"这篇工作提出了 AgenticDataBench，一个面向数据智能体的综合评测基准，用于更细粒度地衡量数据科学工作流中的能力。作者从 15 个垂直领域收集真实数据集和任务，并引入基于技能的覆盖度刻画来去除冗余、提升任务质量。对于缺少真实任务的领域，论文还用 LLM 结合技能体系自动生成更贴近真实场景的工作流任务。实验部分对多种先进数据智能体进行了评测，给出了按技能维度拆解的详细分析，能更清楚地暴露不同系统在真实数据科学流程中的短板。"}
/>

<PaperCard
  title={"Breaking Failure Cascades: Step-Aware Reinforcement Learning for Medical Multimodal Reasoning"}
  arxivUrl={"https://arxiv.org/abs/2606.31825"}
  hjfyUrl={"https://hjfy.top/arxiv/2606.31825"}
  summary={"这篇工作针对医学多模态推理中的失败级联问题，提出了 Step-Aware Reinforcement Learning 方法 MRPO。它不是只看最终答案，而是引入逐步过程奖励，并在最终答案错误时对更早的无效推理步骤施加指数级更大的惩罚，从而打断错误在推理链中的传播。作者在三个多模态 LLM 骨干上验证了该方法，相比标准 GRPO 和近期 RL 基线都有稳定提升。实验中，Qwen3-VL-8B-Instruct 甚至比更大的 HuatuoGPT-Vision-34B 还高出 2.79 分，同时早期推理失败率从 64.0% 降到 13.0%。"}
/>

<PaperCard
  title={"SkillCoach: Self-Evolving Rubrics for Evaluating and Enhancing Agentic Skill-Use"}
  arxivUrl={"https://arxiv.org/abs/2607.01874"}
  hjfyUrl={"https://hjfy.top/arxiv/2607.01874"}
  summary={"这篇论文提出 SkillCoach，用自演化的 rubric 来评估和增强智能体的技能使用能力。方法上，它从真实 rollout 中归纳出与技能相关的过程性评分标准，并从技能选择、技能遵循、技能组合和技能反思四个维度对轨迹进行评价，同时把外部 verifier 作为独立的结果信号保留。作者还把这些演化出来的 rubric 用作训练监督，筛选更高质量的轨迹。实验显示，这种方式能显著提升评测质量，揭示仅看最终准确率无法发现的失败模式，并提供比只做结果过滤更强的训练信号。"}

## Multimodal World Model

<div class="space-y-4">
<PaperCard
  title={"WorldDirector: Building Controllable World Simulators with Persistent Dynamic Memory"}
  arxivUrl={"https://arxiv.org/abs/2607.02517"}
  hjfyUrl={"https://hjfy.top/arxiv/2607.02517"}
  summary={"这篇论文提出 WorldDirector，用于构建具备持久动态记忆和可控视角探索能力的视频世界模型。方法上，它把语义层面的运动编排与视觉生成解耦，先用 LLM 协调 3D 轨迹和相机运动，再把这些轨迹作为控制信号驱动视频生成。这样可以在角色长时间离开画面后再次回到场景时，仍保持精确的身份一致性和外观稳定性。实验表明，该框架能够合成更复杂、更长程的事件，并显著提升生成过程的可控性与动态对象记忆能力。"}
/>
</div>
