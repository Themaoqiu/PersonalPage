## Agent Training and Evaluation

<div class="space-y-4">
<PaperCard
  title={"Retrospective Harness Optimization: Improving LLM Agents via Self-Preference over Trajectory Rollouts"}
  arxivUrl={"https://arxiv.org/abs/2606.05922"}
  hjfyUrl={"https://hjfy.top/arxiv/2606.05922"}
  summary={"这篇论文提出 Retrospective Harness Optimization（RHO），用自监督方式优化 LLM agent 的 harness，而不依赖人工标注的验证集。方法上，它从历史轨迹中挑选具有代表性的困难任务，重新并行求解这些任务，再通过 self-validation、self-consistency 和 pairwise self-preference 选择最优的 harness 更新方案。作者在软件工程、技术工作和知识工作三类场景上验证了方法有效性。实验中，单轮优化就将 SWE-Bench Pro 的通过率从 59% 提升到 78%，并且能持续改善长程会话中的行为模式。"}
/>

<PaperCard
  title={"SearchSwarm: Towards Delegation Intelligence in Agentic LLMs for Long-Horizon Deep Research"}
  arxivUrl={"https://arxiv.org/abs/2606.09730"}
  hjfyUrl={"https://hjfy.top/arxiv/2606.09730"}
  summary={"这篇工作聚焦长程深度研究中的“delegation intelligence”，即主 agent 如何分解任务、委派子任务并整合结果。作者设计了一个 harness 来引导高质量的任务分解与委派，同时约束子 agent 返回适合主流程使用的结果，再把这些轨迹转化为监督微调数据以将委派能力内化到模型参数中。实验显示，训练后的 SearchSwarm-30B-A3B 在 BrowseComp 上取得 68.1、在 BrowseComp-ZH 上取得 73.3，达到同规模模型中的最佳结果。论文还释放了 harness、模型权重与训练数据，便于后续研究。"}
