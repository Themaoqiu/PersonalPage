## Agent Training and Evaluation

<div class="space-y-4">
<PaperCard
  title={"Retrospective Harness Optimization: Improving LLM Agents via Self-Preference over Trajectory Rollouts"}
  arxivUrl={"https://arxiv.org/abs/2606.05922"}
  hjfyUrl={"https://hjfy.top/arxiv/2606.05922"}
  summary={"论文提出 Retrospective Harness Optimization（RHO），用历史轨迹自监督优化 LLM agent 的 harness，而不依赖新的标注验证集。方法先从过去轨迹中选取困难任务子集并并行重解，再结合自验证与自一致性生成候选 harness 更新，最后通过模型自身的 pairwise self-preference 选择最优方案。作者在软件工程、技术工作和知识工作三类场景评测该方法。单轮优化就把 SWE-Bench Pro 的通过率从 59% 提升到 78%，并且分析显示它能持续针对既有失败模式改进长程会话表现。"}
/>

  arxivUrl={"https://arxiv.org/abs/2606.09730"}
  hjfyUrl={"https://hjfy.top/arxiv/2606.09730"}
  summary={"论文研究长程深度检索场景中的 delegation intelligence，提出 SearchSwarm 用于训练 agent 学会任务分解、何时委派以及如何整合子代理结果。作者设计了一个 harness 来引导高质量分解与委派，并让子代理按约束返回摘要结果，从而把真实委派轨迹转成监督微调数据。基于此训练出的 SearchSwarm-30B-A3B 在 BrowseComp 上取得 68.1、在 BrowseComp-ZH 上取得 73.3，达到同规模模型最佳。作者同时释放了 harness、模型权重和训练数据。"}
