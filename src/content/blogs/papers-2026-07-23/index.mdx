<PaperCard
  title={"DataFlow-Harness: A Grounded Code-Agent Platform for Constructing Editable LLM Data Pipelines"}
  arxivUrl={"https://arxiv.org/abs/2607.16617"}
  hjfyUrl={"https://hjfy.top/arxiv/2607.16617"}
  summary={"这篇工作提出了 DataFlow-Harness，用于让 LLM 代理在真实平台中构建可编辑、可持久化的数据处理流水线。它针对传统代码代理只能生成脚本、却难以直接落到平台原生 DAG 这一 NL2Pipeline 缺口，改用带类型约束的增量式 DAG 修改。系统结合了 DataFlow-Skills、暴露实时操作符注册表和流水线状态的 MCP 层，以及同步对话式编写和可视化 DAG 编辑器的 DataFlow-WebUI。作者在 12 个数据工程任务上测得 93.3% 的端到端通过率；相较 Vanilla Claude Code，成本降低 72.5%，生成延迟降低 49.9%。结果还显示，其观测到的通过率仅比 Context-Aware Claude Code 低 0.9 个百分点，但成本仍低 42.8%，说明实时平台 grounding 能更便宜地生成持久、可编辑的工作流产物。"}

<PaperCard
  title={"AgentDebugX: An Open-Source Toolkit for Failure Observability, Attribution, and Recovery in LLM Agents"}
  arxivUrl={"https://arxiv.org/abs/2607.18754"}
  hjfyUrl={"https://hjfy.top/arxiv/2607.18754"}
  summary={"这篇工作提出 AgentDebugX，一个面向 LLM Agents 的开源调试工具链，强调失败观测、归因和恢复的闭环流程。核心方法 DeepDebug 通过多轮根因诊断，结合全局轨迹理解、结构化检索和交叉质询，来定位真正导致失败的步骤，而不是只看报错表面。作者在 Who and When 基准上报告了更高的严格归因准确率，在 qwen3.5-9b 上达到 28.8% 的 exact agent-and-step accuracy。它还在 GAIA 上单次 rerun 修复了 73 个失败任务中的 13 个，把整体准确率从 55.8% 提升到 63.6%。"}
/>

<PaperCard
  title={"Two-Level Meta-Rubrics for Evaluating Open-Ended Generation: GAMUT, a Benchmark for Factual Completeness"}
  arxivUrl={"https://arxiv.org/abs/2607.19322"}
  hjfyUrl={"https://hjfy.top/arxiv/2607.19322"}

## Multimodal World Model

<div class="space-y-4">
<PaperCard
  title={"ABot-World-0: Infinite Interactive World Rollout on a Single Desktop GPU"}
  arxivUrl={"https://arxiv.org/abs/2607.19191"}
  hjfyUrl={"https://hjfy.top/arxiv/2607.19191"}
  summary={"这篇工作提出了 ABot-World-0，一个面向实时、长时程闭环交互的动作条件视频世界模型。方法上，它用来自 AAA 游戏、模拟器和互联网视频的多源数据基础设施训练，并通过 WorldExplorer 主动采集、14 项确定性质量检查、VLM 评估以及同步动作与文本标注来构建数据。模型训练上，作者将双向动作条件教师逐步蒸馏为因果学生，并引入 LongForcing 来对齐长距离自回归滚动，缓解分布偏移和漂移。系统部署上结合轻量 VAE 解码器、高效注意力、内存感知调度和低比特 DiT 推理，实现了在单张 NVIDIA RTX 5090 上以最高 16 FPS 流式生成 720P 视频，首帧动作延迟约 1.2 秒、峰值显存约 19GiB。实验在 WorldRoamBench 和更长的交互滚动中显示出有竞争力的可控性和较连贯的长期世界演化。"}
/>
