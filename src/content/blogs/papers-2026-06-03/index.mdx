## Agent Training and Evaluation

  arxivUrl={"https://arxiv.org/abs/2605.28556"}

  summary={"这篇工作提出 TASTE，用“从工具序列演化生成任务”的方式自动构造更难、覆盖更广的智能体基准。方法上先用带 LLM 质量信号的自适应对比 n-gram 模型采样有效工具序列，再通过聚类挑选代表序列，实例化成完整任务，并迭代提升难度。作者据此构建了 τ^c-Bench 作为 τ^2-Bench 的扩展，在三个领域上评测 11 组 agent/LLM 配对。结果显示，原本接近饱和的模型在新任务上性能明显下滑，例如 Gemini-3-Flash 从 0.82-0.94 降到 0.28-0.61，同时任务中的独特工具组合数翻倍以上。"}
/>

<PaperCard
  title={"K-BrowseComp: A Web Browsing Agent Benchmark Grounded in Korean Contexts"}
  arxivUrl={"https://arxiv.org/abs/2606.02404"}
  hjfyUrl={"https://hjfy.top/arxiv/2606.02404"}
  summary={"这篇工作提出 K-BrowseComp，一个面向韩国语境的网页浏览智能体基准，包含 400 道题，其中 300 道为人工构造并由韩语母语者验证。方法上还构建了 100 道合成诊断集，利用 hard few-shot exemplars 和针对失败模式的生成来提高对抗性。作者在该基准上评测多种前沿模型与韩国本土模型，发现前者在 Verified 子集上也仅有 30.00%-45.67%，而韩国本土模型仅有 0.00%-10.33%。对抗过滤后的合成诊断集上，最强模型也只有 26.00%，说明该基准对网页浏览与推理能力有较强压力测试作用。"}
/>

<PaperCard
  title={"Harness-1: Reinforcement Learning for Search Agents with State-Externalizing Harnesses"}
  arxivUrl={"https://arxiv.org/abs/2606.02373"}

/>

  arxivUrl={"https://arxiv.org/abs/2606.01311"}
